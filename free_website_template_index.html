About this template

This template is intentionally minimal: one file containing HTML, CSS, and small inline images. Use it as a landing page, small portfolio, or mini business site. Replace the logo, change the colors, and add your content.

How to customize

  1. Open index.html in a text editor (VS Code is recommended).
  2. Edit headings, paragraphs, and links. Replace the SVG images with your own images or an <img> tag pointing to a hosted image.
  3. To change theme color, update the #007acc color values in the stylesheet.

Contact

This contact form is a visual placeholder. To make it functional, connect it to a service like Formspree, Netlify Forms, or a server endpoint.
